# Vector Space

Electron + React + TypeScript app skeleton for a desktop library experience.

## Requirements

- Node.js 20+
- npm 10+

## Getting started

```bash
npm install
```

## Run in development

Starts all dev processes:

- TypeScript watcher for Electron `main` and `preload`
- Vite dev server for renderer with hot reload
- Electron process (restarts when main/preload output changes)

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Quality checks

```bash
npm run lint
npm run format:check
npm run typecheck
```
